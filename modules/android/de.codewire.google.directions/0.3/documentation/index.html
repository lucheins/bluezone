googledirections Module

Description

Simple routing module using Google Directions.

Accessing the googledirections Module

To access this module from JavaScript, you would do the following:

var gd = require("de.codewire.google.directions");

The gd variable is a reference to the Module object.

gd.getRoute(/ Object / config)

Retrivies a MapRouteType. For config options, see config property.

gd.getRoutes(/ Object / config)

Retrivies a MapRouteType. For config options, see config property.

gd.config

Possible options are:

Required

  • origin // {String|Object} The origin address, can be either a string or object (see gd.address).
  • destination // {String|Object} The target address, can be either a string or object (see gd.address).
  • callback // {Object} The callback to call upon completion of the request.

Optional

  • mode // {String} one of the gd.travelModes
  • waypoints // {String|Object}[] Array of waypoints (see gd.address).
  • color // {String} The color of the route.
  • name // {String} Name of the route, needed by map.addRoute();. Defaults to '' (empty string).
  • width // {Number} Width of the route stroke. Defaults to 2.
  • language // {String} A language code, one of the gd.languages.
  • useMapsV2 // {Boolean} Set to true if you are using maps v2 api, defaults to false.
  • optimize // {Boolean} Set to true if you want the route to be optimized, works only if you specify multiple waypoints. Defaults to false.
